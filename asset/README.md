# Asset Folder

Place your images in this folder:

## Required Images:

1. **avatar.png** - Your profile picture/avatar (recommended: 400x400px or larger, square format)
   - Used in the hero section
   - If not provided, a placeholder with icon will be shown

2. **about-image.png** - Image for the About section (recommended: 600x400px or similar)
   - Used in the About Me section
   - If not provided, the section will still work without the image

3. **project1.png** - Screenshot for Project 1 (recommended: 600x400px)
   - Used in the Projects section
   - If not provided, a gradient background will be shown

4. **project2.png** - Screenshot for Project 2 (recommended: 600x400px)
   - Used in the Projects section
   - If not provided, a gradient background will be shown

5. **project3.png** - Screenshot for Project 3 (recommended: 600x400px)
   - Used in the Projects section
   - If not provided, a gradient background will be shown

## Notes:

- All images will have fallback styling if they're not found
- The website will work perfectly fine without these images
- You can replace these with your actual project screenshots and photos
- Recommended formats: PNG or JPG
- Optimize images for web to ensure fast loading times
